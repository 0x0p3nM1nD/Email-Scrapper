# Email-Scrapper
This is a little Email Scrapper application, please read the README file to understand more about the application!


This project is a simple and efficient email extraction tool designed for educational and research purposes.
It scans text content or publicly accessible web pages and extracts all valid email patterns using regular expressions.

How to Use

1. Download the email-scraper.py file.
2. Open your terminal and navigate to the folder where the file is saved (e.g., Downloads).
3. Run the script using:
4. python3 email-scraper.py


When the tool starts, simply enter the target URL you want to scan and wait for the extraction to finish.

Example:

cd Downloads
python3 email-scraper.py
[+] Enter target URL to scan: https://randomsite.com
